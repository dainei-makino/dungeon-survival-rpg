# Environment Character Assets

This directory stores JSON vertex data for decorative environment characters.
The files are loaded directly by the game.

-Available character meshes:

- `seaweed-blocky.json`
- `woodPiece.json`
- `apple.json`
- `stump.json`
- `fallenLeaves.json`
- `tree.json`
- `bush.json`
- `stalactite.json`
- `mushroom.json`
- `grass.json`
- `stone.json`
